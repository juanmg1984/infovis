Indicadores
Globales

Exploración interactiva de indicadores del Banco Mundial por país, región y año a través de un gráfico de burbujas dinámico.

Cómo usar esta visualización

  • Seleccioná el indicador que querés ver en el eje X mediante el primer selector (por ejemplo: PBI per cápita, mortalidad infantil, etc.).
  • Elegí el indicador para el tamaño de las burbujas con el segundo selector — a mayor burbuja, mayor valor del indicador.
  • Usá el filtro de región para enfocarte en una zona geográfica o ver todos los países al mismo tiempo.
  • Pasá el cursor sobre cualquier burbuja para ver el nombre del país y los valores exactos.

Acerca de los datos

Los indicadores provienen de la World Bank Data API (WBD), una API REST pública que permite acceder a más de 16.000 series estadísticas sobre desarrollo económico, social y ambiental de más de 200 países.

La API devuelve datos en formato JSON o XML y no requiere autenticación para consultas básicas. Ver documentación oficial →

api.worldbank.org/v2 country/{code}/indicator/{id} format=json per_page=100
