Mapa Mundial
de Indicadores

Visualización geográfica de indicadores del Banco Mundial. Tamaño y color representan variables independientes — explorá dos dimensiones a la vez, en cualquier año y proyección.

Cómo usar esta visualización

  • Elegí el Indicador A (tamaño) y el Indicador B (color) para ver dos variables simultáneamente — por ejemplo, tamaño = Población y color = PIB per cápita.
  • Usá el slider de año para ver cómo evolucionaron los indicadores entre 2000 y 2023. Los datos se actualizan automáticamente desde la API del Banco Mundial.
  • Cambiá la proyección del mapa (Equal Earth, Mercator, Orthographic, etc.) para explorar cómo cada una distorsiona el espacio geográfico.
  • Filtrá por región para enfocarte en una zona del mundo y atenuar el resto.
  • Hacé clic en cualquier burbuja para resaltar todos los países de esa región. Clic de nuevo para volver a la vista global.
  • Pasá el cursor sobre un país para ver sus valores, año del dato y ranking mundial en el indicador de tamaño.
